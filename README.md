# Game-JAM-2026

This repo now contains the first Love2D prototype slice for a Tomb of the Mask inspired game.

Run it from the `game/` folder with Love2D:

```bash
love .
```

Current prototype features:

- Grid-based movement
- Wall rendering using the existing `game/assets/Wall_1-1.webp` asset
- Glitch phases that can invert controls, boost movement, or shift the palette
- `R` to reset the prototype
